## About The Project

<div align="center">
<img src="images_readme/1.png" alt="1" width="900" height="600">
</br>

Esta app fue elaborada en la materia de Entornos de Desarrollo en el curso de Desarrollo de aplicaciones multiplataforma 2019-2020. About
Simula una app donde compras productos de amazon y los añades desde la url de Amazon.
En ella puedes añadir productos apartir de la url de amazon, contiene un scrapper que extrae los datos necesarios, posteriormente te lo añade al carrito y guarda las compras. Todo esta gestionado por el componente Navigation y sus Actions.
Y todo se guarda en una base de datos SQlite.
</br>

## Usage

Selecione un producto de amazon y luego pegelo en añadir producto, y tendrá el producto. Luego podrá comprar y añadir cosas a su carrito para su posterior compra y se almacenará los pedidos.
